# Canonical Services

This directory contains services for canonical document persistence and retrieval as defined in Plan Suite v2.

## Services

- **CanonicalDocumentService** - Document upsert and retrieval (EP-02)
- **CanonicalChunkService** - Chunk persistence and retrieval (EP-02)
- **EmbeddingService** - Embedding generation and persistence (EP-07)
