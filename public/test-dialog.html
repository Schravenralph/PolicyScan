Dialog Verification Test Page

Purpose: This page is used to verify that dialogs have light, non-transparent backgrounds.

Note: The actual dialog component is rendered by the React app. To see the dialog, you need to:

  1. Navigate to the main app
  2. Trigger a draft reconciliation scenario
  3. Or use the browser console to manually show the dialog

Screenshot Location:

screenshots/dialog-verification.png

To manually trigger the dialog in the app, open the browser console and run: 

// Set up mock draft divergence
localStorage.setItem('beleidsscan-draft', JSON.stringify({
  step: 2,
  queryId: '697f316c36f28245a24687c6',
  onderwerp: 'huisvesting arbeidsmigranten',
  timestamp: new Date().toISOString()
}));

// Reload the page to trigger reconciliation
window.location.reload();
